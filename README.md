# Autils

Some Utils For Android this is test

一些android工具类的汇总

# gradle

Add it in your root build.gradle at the end of repositories:
```
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
Add the dependency:
```
dependencies {
	        compile 'com.github.miaoyongjun:AUtils:1.0.1'
	}
```
